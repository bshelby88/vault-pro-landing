# Anthropic Discord #show-and-tell post — Vault Pro

Hey folks 👋

Built Vault Pro — Obsidian vault pre-structured for Claude Code operation. Bridges the agent-stateless / vault-static gap.

8 folders, functional day 1:
• 00-Start — Dashboard with live Dataview queries
• 20-Agents — 3 runnable Hermes multi-agent specs
• 30-Daily — Daily + monthly review templates (EOD reflection)
• 50-Workflows — 4 orchestration patterns + lead-gen pipeline
• 60-Knowledge — ICP framework + retrospectives folder
• Plus skill index, project tracker, note templates

Claude reads the vault as working memory. Fill the daily template, the monthly review auto-rolls (we wired a recurring agent that drafts the prior-month review on the 1st).

Install: `unzip` → open as Obsidian vault. ~30 sec. Templater + Dataview recommended.
